Overview of describing formal systems in plain English,
i.e. Context for Godel’s Completeness and Incompleteness theorems

https://en.wikipedia.org/wiki/Formal_system

A formal system can be denoted by:

  • A finite set of symbols, i.e. alphabet.
  • A grammar specifying rules for forming well-formed finite sequences (formulas) from simpler well-formed finite sequences of symbols from the alphabet. It is very useful (and often required) to have a decision procedure for whether a finite sequence, is a well-formed formula.
  • A set of axioms, or axiom schemata, consisting of well-formed formulas.
  • A set of inference rules. Theorems are defined as well-formed formulas, that are inferrable from the axioms.

A formal system is said to be:

  • recursive (alternately, effective) if the set of axioms and the set of inference rules are decidable (the algorithm recognizing the axioms, and the algorithm recognizing the inference rules, both terminate in finite time both when the proposed finite sequence does, and does not, match)
  • recursively enumerable if the set of axioms and the set of inference rules are semidecidable (the algorithm recognizing the axioms, and the algorithm recognizing the inference rules, both terminate in finite time when the proposed finite sequence does match; does not have to terminate when it does not match.)

Typical logical symbols for an alphabet include

  • The universal and existential quantifiers.
  • Logical symbols for “and”, “inclusive or”, “not”, “implies”, and “if and only if”.
  • Various kinds of punctuation.
  • An infinite set of variables; suffixing (e.g, with integers) is an option. Their values are in the domain of discourse.
  • Equality = may be considered either a logical symbol, or a non-logical predicate symbol.
  • Truth values.

Typical non-logical symbols are generally either

  • n-ary predicates, that evaluate to truth values. A 0-ary predicate is a propositional variable that can be substituted out for any statement.
  • n-ary functions, that evaluate to objects in the domain of discourse. A 0-ary function is a constant symbol.

Fixing a deductive system (e.g., classical logic):

We say that a sentence s of a theory T:

  • Is a syntactic consequence T├ s , if and only if s is provable from T .
  • Is a semantic consequence T╞ s, if and only if s holds in every model i.e. interpretation M of T .

Terms (symbols that evaluate to an object in the universe of discourse) and formulas (symbols that evaluate to a truth value) may be recursively defined as follows:

  • Any variable, is a term.
  • Any n-ary predicate, taking a non-negative integer n terms, is a formula.
  • Any n-ary logical connective, taking a positive integer n formulas, is a formula.
  • For any formula φ and variable x, the quantified statements Ax φ and Ex φ are formulas.
  • Any n-ary function symbol f, taking a non-negative integer n number of formulas or terms as arguments, is a term.
    • This is atypical, but needed to be able to denote a step function in first-order logic.

Bound and free variables in formulas are defined as follows:

  • There are no bound variables in any atomic formula; a variable x is free in an atomic formula φ, if and only if x occurs in φ
  • For any n-ary logical connective, a variable x is free or bound in that connective, if and only if x is free or bound in at least one formula of that connective.
  • For formulas Ax φ and Ex φ, a variable
    • y is free if and only if y<>x and y is free in φ
    • y is bound if and only if y=x or y is bound in φ
      • We do not mention free variables in the definition of bound variables, and vice versa, so that the definitions work as expected in nonclassical logics.

Specific formal systems may introduce additional base case formulas with comparable definitions of bound and free variables (e.g., the class operator in Von Neumann-Bernays-Gödel set theory).

A formula in first-order logic, that contains no free variables, is called a first order sentence. These are the formulas for which it is reasonable to expect them to evaluate to truth values.

If a sentence φ evaluates to true for a given interpretation M of a theory T, we say M satisfies φ ; symbolic denotation is M╞ φ . A sentence φ is satisfiable if there is at least one interpretation i.e. model M for which φ is true.

A formula that does contain at least one free variable, may be defined to be satisfiable if it is true when any element of the universe of discourse is substituted for the free variable. Whether this definition is useful, is context-sensitive.

A formula φ is said to be (logically) valid, if it is true in every interpretation i.e. model M. (This is closely related to semantic consequence T╞ φ .)

A formula φ is said to be a logical consequence of a formula ψ, if every interpretation that makes ψ true also makes φ true.

In (full) second-order logic, we allow variables that range over k-ary relations (where k is a positive integer). Monadic second-order logic only uses variables that range over 1-ary relations, i.e. sets.

Standard semantics is the “naive” generalization of first order logic to second order logic; Henkin semantics involves domain restrictions for the second-order logic specific variables, compared to standard semantics.
